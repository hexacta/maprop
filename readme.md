## Getting started

### First run

On `back` run:
```bash
$ npm install
```

On `front-react` run:
```bash
$ npm install
```

### Running backend

On `back` run:
```bash
$ node index.js
```

Server will run on [http://localhost:4003](http://localhost:4003)

### Running client

On `front-react` run:  
```bash
$ yarn start
```

Client will run on [http://localhost:3000](http://localhost:3000)

## Deploy

Every commit to master is deployed to github pages using travis.
